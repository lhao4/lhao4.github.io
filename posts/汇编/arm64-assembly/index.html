1. 核心概念（必须知道）#

1.1 寄存器（32个通用寄存器）#

寄存器用途记忆口诀
X0-X7函数参数/返回值0-7传参数
X8系统调用号8号 syscall
X9-X15临时变量（用完即弃）临时9到15
X16-X17链接器专用链接1617
X18平台保留平台18
X19-X28跨函数保存的变量保存19-28
X29 (FP)栈帧指针29帧指针
X30 (LR)返回地址30要返回

W0-W30：X寄存器的低32位版本。用W操作后，高32位自动清零。

1.2 特殊寄存器#

名称功能关键指令
SP栈指针STP/LDP 操作栈
PC程序计数器（当前执行位置）自动+4
XZR零寄存器（读=0，写=丢弃）CMP 底层用它

2. 最常用的指令（先学这20个）#

2.1 数据搬运#

指令示例含义
MOVMOV X0, X1X0 = X1
MOVMOV X0, #100X0 = 100
LDRLDR X0, [X1]X0 = *X1（从内存读）
STRSTR X0, [X1]*X1 = X0（写内存）
LDPLDP X0, X1, [SP]从栈弹出两个值
STPSTP X0, X1, [SP, #-16]!压栈两个值（!表示先减SP）

方括号[] = 内存访问

2.2 算术运算#

指令示例含义
ADDADD X0, X1, X2X0 = X1 + X2
SUBSUB X0, X1, #10X0 = X1 - 10
ADDSADDS X0, X1, X2加法+更新标志位
CMPCMP X0, X1比较X0-X1（结果丢到XZR）

带S的指令会更新NZCV标志位

2.3 逻辑与移位#

指令示例含义
ANDAND X0, X1, X2按位与
ORRORR X0, X1, X2按位或
LSLLSL X0, X1, #3左移3位（×8）
LSRLSR X0, X1, #2右移2位（÷4）

2.4 分支跳转#

指令示例含义
BB label无条件跳转
B.EQB.EQ label相等则跳转（还有其他条件）
CBZCBZ X0, labelX0==0则跳转
CBNZCBNZ X0, labelX0!=0则跳转
BLBL func调用函数（保存返回地址到X30）
RETRET返回（从X30取地址）

条件后缀：EQ(=), NE(≠), GT(>), LT(<), GE(≥), LE(≤)

2.5 系统调用#

指令示例含义
SVC #0SVC #0触发系统调用

系统调用模板：

MOV X0, fd      ; 第1个参数
MOV X1, buf     ; 第2个参数  
MOV X2, count   ; 第3个参数
MOV X8, #63     ; syscall号（read=63）
SVC #0          ; 进入内核

3. 函数调用约定（AAPCS64）#

3.1 参数传递#

  • 整数/指针：X0-X7（最多8个）
  • 浮点：V0-V7
  • 更多参数：压栈

3.2 返回值#

  • X0：主返回值
  • X1:X0：128位返回值（高位:低位）

3.3 寄存器保存责任#

谁保存寄存器
调用者保存（调用前自己压栈）X0-X18
被调用者保存（函数内用完要恢复）X19-X28, X29, X30

4. 典型代码模式（看懂这些就够了）#

4.1 函数序言/尾声（标准套路）#

; 函数开头（Prologue）
STP X29, X30, [SP, #-16]!   ; 保存FP和LR，分配16字节栈空间
MOV X29, SP                  ; 更新帧指针

; ... 函数体 ...

; 函数结尾（Epilogue）  
LDP X29, X30, [SP], #16      ; 恢复FP和LR，释放栈空间
RET                          ; 返回

看到 STP X29, X30 就知道是函数开头，看到 LDP X29, X30 + RET 就是函数结尾

4.2 条件判断#

CMP X0, X1        ; 比较
B.GT greater      ; 有符号大于则跳
B.HI higher       ; 无符号大于则跳（注意区别！）

; 或者无分支版本
CSEL X2, X0, X1, GT   ; X2 = (X0>X1) ? X0 : X1

4.3 循环结构#

    MOV X0, #10       ; 计数器
loop:
    CBZ X0, done      ; X0==0则结束
    SUB X0, X0, #1    ; 计数器减1
    B loop            ; 继续循环
done:

4.4 访问全局变量（PIC代码）#

ADRP X0, sym_page       ; 加载4KB页基地址（±4GB范围）
ADD  X0, X0, :lo12:sym  ; 加上页内偏移
LDR  X1, [X0]           ; 读取全局变量

5. 快速查表：条件码#

条件含义测试
EQ等于Z==1
NE不等于Z==0
GT大于（有符号）Z0 && NV
LT小于（有符号）N!=V
HI高于（无符号）C1 && Z0
LO低于（无符号）C==0

6. 实战：读一段汇编#

C代码：

int add(int a, int b) {
    return a + b;
}

ARM64汇编：

add:
    ADD W0, W0, W1    ; W0 = W0 + W1（用W因为int是32位）
    RET               ; 返回，结果在W0

C代码：

int max(int a, int b) {
    return (a > b) ? a : b;
}

ARM64汇编：

max:
    CMP W0, W1        ; 比较a和b
    CSEL W0, W0, W1, GT   ; 如果GT(大于)，W0=a，否则W0=b
    RET

7. 学习路径建议#

第1周：记住寄存器用途 + MOV/LDR/STR/ADD/SUB
第2周：掌握分支指令 B/CBZ/CMP + 条件码
第3周：理解函数调用 BL/RET + 栈操作 STP/LDP
第4周：系统调用 SVC + 看实际代码

推荐练习：

  1. 用 objdump -d 反编译简单C程序
  2. 对照本手册逐行理解
  3. 修改C代码，观察汇编变化
