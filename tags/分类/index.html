基于逻辑回归的鸢尾花分类项目研究报告

1 任务描述 1.1 项目背景 鸢尾花分类是机器学习领域的经典多分类任务，常作为入门级案例用于验证分类算法的有效性。本项目以逻辑回归为核心算法，针对鸢尾花数据集完成多分类任务，重点对比L1、L2正则化策略对模型性能、复杂度及可解释性的影响，并通过多维度可视化分析揭示模型特性与数据规律。 1.2 任务目标 基于Julia语言手动实现支持多分类的逻辑回归（Softmax回归）模型； 系统对比无正则化、L1正则化、L2正则化三种配置下模型的训练效果与泛化能力； 完成特征重要性分析、决策边界可视化等可解释性分析； 构建完整的模型评估体系，并提出针对性的改进方案。 1.3 技术路线 本项目遵循“数据探索→预处理→模型设计→训练评估→可视化分析→结论优化”的机器学习全流程，技术栈包括Julia的DataFrames（数据处理）、PyPlot（可视化）、LinearAlgebra/Statistics（数值计算）等库，核心是手动实现带正则化的逻辑回归训练逻辑。 2 数据集分析 2.1 数据集概述 鸢尾花数据集（Iris Dataset）包含150个样本，无缺失值，是机器学习领域的基准数据集。核心信息如下： 样本规模：150个（3类鸢尾花，每类50个样本）； 特征维度：4个连续数值型特征（花萼长度/宽度、花瓣长度/宽度，单位：cm）； 类别标签：Setosa（0）、Versicolor（1）、Virginica（2）； 数据质量：无缺失值、无异常值，特征均为浮点型，无需额外清洗。 2.2 样本分布与数据探索 2.2.1 类别分布 三类鸢尾花样本数量均等（各50个，占比33.3%），无类别不平衡问题，为模型训练提供了均衡的样本基础。 2.2.2 特征分布与相关性分析 通过综合数据探索可视化（图2-1）可得出核心规律： 特征分布：Setosa类别的花瓣长度/宽度显著小于另外两类，具备强区分度；Versicolor与Virginica在花萼特征上存在明显重叠； 特征相关性：花瓣长度与花瓣宽度的相关系数达0.96（高度正相关），花萼宽度与其他特征相关性较弱； 散点分布：花瓣特征（长度+宽度）对类别区分的贡献远高于花萼特征。 图2-1 鸢尾花数据集探索性分析图 注：左上至右依次为4个特征的类别分布直方图、类别占比饼图、特征相关性热图、花瓣特征散点图 2.3 数据预处理 2.3.1 数据集分割 按8:2比例随机划分训练集（120个样本）和测试集（30个样本），并固定随机种子（seed=123）保证可复现性。 2.3.2 特征标准化 对特征进行Z-score标准化（均值归零、方差归一化），消除量纲影响： 其中、分别为训练集特征的均值和标准差，标准化后训练集特征均值接近0、标准差接近1，提升模型收敛效率。 3 模型设计 3.1 逻辑回归（Softmax回归）理论基础 针对多分类任务，采用Softmax回归扩展二分类逻辑回归，核心原理如下： 3.1.1 Softmax函数 将线性输出映射为类别概率，保证所有类别概率和为1： 其中：为第类的权重向量，为偏置项，为输入特征向量，（类别数）。 3.1.2 交叉熵损失函数 衡量预测概率与真实标签的差异，是多分类任务的核心损失： 注：添加避免数值溢出；为真实标签的One-hot编码，为预测概率。 3.2 正则化策略设计 为缓解过拟合，引入L1、L2正则化，核心是在损失函数中加入权重惩罚项： 3.2.1 L2正则化（岭回归） 惩罚权重的平方和，使权重均匀收缩，避免单个权重过大： ...
