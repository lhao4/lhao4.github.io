第一篇文章：HaoNest 搭建记录

这是 HaoNest 的第一篇文章，用于测试博客的各项功能。 为什么搭建个人博客 记录学习过程是一种很好的学习方式。个人博客不仅可以整理笔记，还可以分享给他人。 技术选型 静态站点生成器 选择了 Hugo，原因如下： 构建速度极快 无需运行时依赖 丰富的主题生态 Markdown 写作体验好 主题 使用 PaperMod，简洁清晰，功能完备。 部署 部署在 GitHub Pages，完全免费，通过 GitHub Actions 自动构建发布。 站点功能 本站除了博客文章，还集成了： 网站导航 收藏了常用的开发工具、AI 工具、学习资源等网站，按类别整理，方便快速访问。 在线工具 内置多种开发常用工具，包括： 进制转换 JSON / XML / SQL 格式化 正则表达式测试 JWT 解析 时间戳转换 以及更多… 后续计划 持续更新学习笔记 完善工具页面 接入评论系统 感谢访问 HaoNest！
